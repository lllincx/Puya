# ACA

1. 配置aca寄存器
2. 读写TrustEngine寄存器，测试apb功能
3. 配置aca通用寄存器，配置写指针，配置通用寄存器长度，源数据写入sram
4. 将初始化指令写入入口寄存器
5. 开始执行，计算完成后产生中断
6. 进入中断后，读写TrustEngine寄存器，测试apb功能
7. 读取期望存储计算结果的位置并与预期结果比较

# OTP_varibale

1. 读取flash中OTP Page 5~7 数据，并打印flash内容
2. 从读取TrustEngine读取OTP Page 5~7对应的内容，检查与flash中读取数据是否一致
3. 编程flash中OTP Page 5~7 数据，并打印编程的内容
4. 读取flash中OTP Page 5~7 数据，检查与编程后预期是否一致
5. 复位，将Page 5~7 数据载入TrustEngine
