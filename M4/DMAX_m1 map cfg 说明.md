

### 目前配置

```verilog
(axaddr[31:30] = 2'b00)&&(axaddr[27:19] == 9'b0)
```

### 代码解释

- `axaddr[31:30] == 2'b00` → 地址在 0x0000_0000–0x3FFF_FFFF 内
- `axaddr[27:19] == 9'b0` → 每组内只允许偏移 0x0_0000–0x7_FFFF（bits 18:0 自由）
- bits 29:28 无约束 → 四个组全部命中，对应关系

| 组 (bits 29:28) | 合并后范围                  |
| --------------- | --------------------------- |
| 00              | `0x0000_0000 – 0x0007_FFFF` |
| 01              | `0x1000_0000 – 0x1007_FFFF` |
| 10              | `0x2000_0000 – 0x2007_FFFF` |
| 11              | `0x3000_0000 – 0x3007_FFFF` |
